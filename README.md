SmartFill Privacy Policy

Last Updated: March 2026

SmartFill ("SmartFill", "we", "us", or "our") provides a browser extension designed to help users autofill web forms using locally stored profile information. This Privacy Policy explains how information is handled in connection with the SmartFill browser extension.
SmartFill is built with a privacy-by-design architecture. We do not operate servers that collect user data, and we do not process personal information outside the user's device.

1. Scope
This Privacy Policy applies solely to the SmartFill browser extension and describes how user information is handled when using the Extension.

2. SmartFill does not:

 - Collect personal information
 - Transmit user data to external servers
 - Store user data in cloud systems
 - Use analytics or tracking technologies
 - Use advertising frameworks
 - Sell, rent, trade, or monetize user data
 - Share information with third parties
 - We do not have access to your personal information.

4. Local Storage of Data
All information entered into SmartFill — including but not limited to:
 - Full name
 - Parent names
 - Identification numbers (PAN, Aadhaar, Passport, DL, etc.)
 - Addresses
 - Education details
 - Contact information
is stored exclusively in your browser’s local storage on your device.
 - This data:
Remains under your control
Is not transmitted to SmartFill
Is not accessible to us
Is not backed up to external systems
SmartFill does not have the technical capability to access, retrieve, or view your locally stored data.

   4.1 Profile Management
   - The Extension’s popup and settings interface access locally stored data to allow you to:
   - Create profiles
   - Edit information
   - Delete profiles
   - Select profiles for autofill

   4.2 Form Autofill
   - When you activate autofill:
   - The Extension inserts selected profile data directly into form fields on the active webpage.
   - The process occurs entirely within your browser.
   - No data is transmitted to SmartFill or any external service.

5. How the Extension Uses Data
SmartFill uses your locally stored information strictly for functionality within your browser.

6. Permissions
SmartFill may request permission to access websites in order to:
 - Detect form fields
 - Enable autofill functionality
 - Provide contextual field suggestions
 - This access is used solely to deliver core Extension features.
 
   6.1 SmartFill does not:
    - Monitor browsing activity
    - Collect website content
    - Capture keystrokes
    - Track user behavior
    - Transmit browsing data

   6.2 Deleting Profiles
    - When you delete a profile:
    - It is permanently removed from browser local storage.
    - No backup or copy is retained.
 
   6.3 Uninstalling the Extension
    - When you uninstall SmartFill:
All associated local storage data is removed from your browser.
SmartFill retains no copies of your data.

7. User Control and Data Deletion
You retain full control over your data at all times.

8. Third-Party Services
 - SmartFill does not integrate with:
 - Analytics providers
 - Advertising networks
 - Data brokers
 - Cloud storage providers
 - External data processors
The Extension operates entirely within your browser environment.
